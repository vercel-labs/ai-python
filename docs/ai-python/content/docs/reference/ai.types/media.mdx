`ai.types.media` contains URL, data URL, media type inference, and magic-byte
detection helpers.

## URL helpers

- `is_url`
- `is_downloadable_url`
- `split_data_url`

## Encoding helpers

- `data_to_base64`
- `data_to_data_url`

## Media type helpers

- `infer_media_type`
- `detect_media_type`
- `detect_image_media_type`
- `detect_audio_media_type`
