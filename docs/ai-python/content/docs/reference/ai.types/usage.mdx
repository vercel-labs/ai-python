`Usage` is normalized token usage from a single model call.

```python
usage.input_tokens
usage.output_tokens
usage.reasoning_tokens
usage.cache_read_tokens
usage.cache_write_tokens
usage.raw
usage.total_tokens
```

Use `usage_a + usage_b` to accumulate usage across calls.
