`AnthropicCompatibleProvider` implements `Provider` for Anthropic-compatible
APIs.

Default configuration uses:

- `ANTHROPIC_API_KEY`
- `ANTHROPIC_BASE_URL`

Pass `base_url`, `api_key`, or a custom client through `get_provider` when you
need explicit configuration.
