Error helpers expose normalized HTTP context and status-code mapping.

## HTTPErrorContext

```python
context.status_code
context.request
context.response
```

## http_status_to_provider_status_error_class

```python
error_cls = ai.errors.http_status_to_provider_status_error_class(429)
```

Returns the provider status error class for an HTTP status code.
