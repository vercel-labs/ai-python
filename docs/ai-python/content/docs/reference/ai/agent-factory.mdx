`agent` creates an `Agent`.

```python
agent = ai.agent(tools=[contact_mothership])
```

## Arguments

- `tools`: Optional `AgentTool` values from `@ai.tool` and schema-only `ai.Tool`
  declarations for provider-executed tools.

## Return value

Returns `ai.Agent`.
