Routing params configure provider or gateway routing behavior.

## Types

- `RoutingParams`
- `RoutingTarget`
- `RoutingTargetChain`
- `GeoRegion`
- `CloudRegion`
- `ProviderRankingStrategy`

Use `GLOBAL` for global routing when supported by the provider.
