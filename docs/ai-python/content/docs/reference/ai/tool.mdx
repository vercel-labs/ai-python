---
title: "ai.Tool"
description: "Declare a schema-only or provider-executed tool."
type: reference
summary: "Reference for ai.Tool."
---

`Tool` is the model-facing tool declaration used by providers.

```python
tool = ai.Tool(
    spec=ai.tools.ToolSpec(
        name="lookup",
        description="Look up a value.",
        input_schema={"type": "object", "properties": {}},
    )
)
```

Pass `Tool` values to `ai.stream(..., tools=[...])` or to an agent when the
provider should receive the declaration directly. For Python callables, use
`@ai.tool` instead.
