Async-generator tools can yield partial output while they run. The agent emits
`ai.events.PartialToolCallResult` events for yielded values.

## Built-in aliases

- `StreamingTextTool`: Concatenate yielded strings.
- `StreamingStatusTool[T]`: Treat intermediate yields as status updates and the
  last yielded value as the final result.
- `SubAgentTool`: Forward nested agent events and use the nested final text as
  model input.
