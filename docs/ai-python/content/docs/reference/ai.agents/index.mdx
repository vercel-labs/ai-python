Most agent APIs are documented as top-level `ai` exports. Use `ai.agents`
for advanced agent namespace APIs and types that are not promoted to the
top-level namespace.

Public advanced APIs:

- `Aggregate`
- `SimpleAggregator`
- `ConcatAggregator`
- `LastAggregator`
- `MessageAggregator`
- `BoundToolCall`
- `GatedToolCall`
- `ToolCallCallable`
